# UX / UI

## Principes
- 3 taps max pour les actions fréquentes
- iPad-first : split view, large targets, usage intensif
- Offline visible : statuts sync, files d’attente, reprise automatique
- Quick actions + favoris + modèles

## Navigation
- Sidebar gauche (modules principaux)
- Vue principale en split (liste/détail)
- Actions contextuelles persistantes

## Design system
- Palette claire, contrastes élevés, lisible en chantier
- Typo expressive + hiérarchie forte
- États explicites (offline, conflit, upload, signature)
